# Healthcare-Analytics--PowerBI-Dashboard
Healthcare Analytics Dashboard built using Power BI to analyze patient trends and operational metrics. The project demonstrates end-to-end BI skills including ETL with Power Query, data modeling with relationships, and DAX measures for KPIs. Designed to simulate real-world healthcare reporting and decision-support analysis.
The objective of this project is to simulate a real-world healthcare analytics use case and highlight practical BI skills that are commonly used in analyst and reporting roles.

---

## Tools & Technologies
- Power BI Desktop  
- Power Query (ETL)  
- DAX (Data Analysis Expressions)  
- Data Modeling (Relationships)  
- Excel (Data Source)

---

## ETL & Data Preparation
Data was extracted from Excel and transformed using Power Query. The ETL process included:
- Cleaning and standardizing data
- Handling missing or inconsistent values
- Correcting data types
- Renaming columns for clarity
- Creating derived columns required for analysis

These steps ensured data accuracy and consistency before loading it into the data model.

---

## Data Modeling
A structured data model was created using fact and dimension tables. Proper relationships were defined using one-to-many cardinality and single-direction filtering. This approach supports accurate aggregations and efficient DAX calculations while following best practices such as star schema principles.

---

## Key Metrics & Insights
The dashboard provides insights into:
- Patient volume trends
- Department-wise analysis
- Time-based performance using date filters
- Key operational healthcare indicators

Dynamic DAX measures were created to enable interactive analysis across different dimensions.

---

## Outcome
This project demonstrates the ability to build a complete Power BI solution, covering ETL, data modeling, DAX, and dashboard design. It reflects practical healthcare analytics scenarios and is intended for learning, portfolio presentation, and skill demonstration purposes.

Note- The dataset used in this project is a simulated dataset created solely for demonstration and learning purposes.
